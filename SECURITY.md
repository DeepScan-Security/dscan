# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in dscan, please do NOT
open a public GitHub issue.

Email: security@deepscan.security
Response time: within 48 hours

We will acknowledge receipt, investigate, and coordinate a fix
before public disclosure. We follow responsible disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | ✅ Yes    |

## Scope

dscan is a security tool. We take vulnerabilities seriously.
Especially relevant: injection in the redactor, trace file
path traversal, dashboard authentication bypass.
